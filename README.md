
# Beatstick

Dies ist ein Programm geschrieben von Jan-Luca Müller, Jonas Volkmann und Cedric Schiller für das Modul HCI an der HSB.

## Features

- 4 Instrumente
- Instrument-Sample wechseln
- Start/Stop
- Aus/An
- Tempo ändern
- Sequencer-Länge ändern
- Motion-Control
- 3D-Viewer
- Tutorial


## Roadmap

- Motion-Control für IOS

- Mehr Instrumente
